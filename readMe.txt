# DhakaNest AI 🏠🤖

**DhakaNest AI** is an agent-based rental assistant designed for dense cities like Dhaka. It understands user needs (budget, location, lifestyle) and automatically searches, filters, and recommends suitable flats or rooms. Built to solve real student housing challenges, it simplifies and speeds up the entire house-hunting process.

## 🚀 Key Features
- **AI-Driven Search:** Intelligent filtering based on lifestyle and budget.
- **Dual Interface:** Separate modules for **Tenants** looking for homes and **Owners** listing properties.
- **Authentication:** Secure login and user management system.
- **Dynamic Navigation:** Seamless transitions between search results and property details.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Logic:** Custom Search Algorithms and Agent-based modules.
- **Data:** JavaScript-based database management.

## 📂 Project Structure
- `index.html`: Main entry point.
- `search-algorithm.js`: Core AI logic for matching tenants with flats.
- `auth.js`: Handles user registration and login.
- `tenant.js` / `owner.js`: Role-specific functionalities.
- `database.js`: Mock database for property and user records.
